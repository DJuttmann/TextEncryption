-- DESCRIPTION --
A program to encrypt and decypt text messages with Caesar cipher or Vigenère cipher. Text
can be loaded from and saved to text files.

-- USAGE --
Write/Load text you want to encrypt in the 'plaintext' box, or encrypted text in the
'ciphertext' box. Select the encryption scheme from the drop-down menu and enter the key
in the box below. Press encrypt/decrypt buttons to show the encrypted/decrypted text.
Plaintext and ciphertext can be saved to a text file as well.
